# 📚 4-Week Learning Curriculum: PYTHON PROGRAMMING

## 🔒 Safety Checklist & Guidelines
Before beginning, please read and follow these safety guidelines:

  1. Always run code in a virtual environment or isolated container to protect your system files.
  2. Do not execute unknown or untrusted scripts from the internet.
  3. Keep your Python installation updated to the latest stable release for security patches.
  4. Ensure your computer has sufficient storage for development tools and IDEs.
  5. Take regular breaks to avoid digital eye strain.

## 📅 Week 1: Understand Python syntax, data types, and basic control flow.

### 📝 Learning Activities

  - Install Python and an IDE like VS Code or PyCharm.
  - Learn about variables, data types, and basic arithmetic operations.
  - Practice using conditional statements and basic loops.

### 🔬 Practical Labs & Simulations

  - Create a script that takes user input and performs simple arithmetic calculations.

### 🔗 Curated Resources

  - **Python Tutorial (Official Python Documentation)** (Documentation): [https://docs.python.org/3/tutorial/index.html](https://docs.python.org/3/tutorial/index.html)

## 📅 Week 2: Deepen knowledge of data structures and functional programming concepts.

### 📝 Learning Activities

  - Study list, tuple, dictionary, and set usage.
  - Learn how to define functions and pass arguments.
  - Understand modules, libraries, and how to import external packages.

### 🔬 Practical Labs & Simulations

  - Build a simple calculator module that includes various mathematical functions.

### 🔗 Curated Resources

  - **Programming for Everybody: Python (Coursera)** (Course): [https://www.coursera.org/specializations/python](https://www.coursera.org/specializations/python)

## 📅 Week 3: Explore advanced concepts including error handling and file manipulation.

### 📝 Learning Activities

  - Practice using try-except blocks for effective error handling.
  - Learn to read from and write to text and CSV files.
  - Review object-oriented programming (OOP) basics such as classes and objects.

### 🔬 Practical Labs & Simulations

  - Create a script that reads user names from a text file and saves them to a structured log file with error handling.

### 🔗 Curated Resources

  - **Python for Beginners - Full Course (YouTube)** (Video): [https://www.youtube.com/watch?v=mock_python_beginners](https://www.youtube.com/watch?v=mock_python_beginners)

## 📅 Week 4: Integrate learned skills into a comprehensive capstone project.

### 📝 Learning Activities

  - Design a project scope involving data management or automation.
  - Write clean, documented, and modular code.
  - Debug and test your final project application.

### 🔬 Practical Labs & Simulations

  - Build a personal task manager application that stores data in a JSON file and includes add, delete, and list functionality.

### 🔗 Curated Resources

  - **Python Tutorial (Official Python Documentation)** (Documentation): [https://docs.python.org/3/tutorial/index.html](https://docs.python.org/3/tutorial/index.html)
